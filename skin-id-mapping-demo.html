Skin ID Mapping System

Overview

This mapping system converts between the old skin IDs (used in index.html) and the new skin IDs (used in skinData.js).

The old system used simple identifiers like flame, dog, etc., while the new system uses more descriptive kebab-case names like hot-head, good-boy, etc.

Usage Examples

// Convert old ID to new ID
const newId = skinIdConverter.toNewId('flame');  // Returns: 'hot-head'

// Convert new ID to old ID
const oldId = skinIdConverter.toOldId('hot-head');  // Returns: 'flame'

// Convert arrays of IDs
const oldIds = ['flame', 'dog', 'neko'];
const newIds = skinIdConverter.toNewIds(oldIds);  // Returns: ['hot-head', 'good-boy', 'lil-beans']

// Check ID types
skinIdConverter.isOldId('flame');     // Returns: true
skinIdConverter.isNewId('hot-head');  // Returns: true

// Get all IDs
const allOldIds = skinIdConverter.getAllOldIds();
const allNewIds = skinIdConverter.getAllNewIds();

Migration Helper

When migrating from the old system to the new system, use this pattern:

// Example: Migrating saved skin preferences
const savedSkinId = localStorage.getItem('selectedSkin');
if (savedSkinId && skinIdConverter.isOldId(savedSkinId)) {
    // Convert and save the new ID
    const newSkinId = skinIdConverter.toNewId(savedSkinId);
    localStorage.setItem('selectedSkin', newSkinId);
    console.log(`Migrated skin: ${savedSkinId} â†’ ${newSkinId}`);
}

// Example: Loading skin data with backward compatibility
function getSkinData(skinId) {
    // Convert old ID to new if necessary
    const normalizedId = skinIdConverter.isOldId(skinId) 
        ? skinIdConverter.toNewId(skinId) 
        : skinId;
    
    return window.SKIN_DATA[normalizedId];
}
